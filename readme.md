## Как запустить

1. ``cd react-implementation`` (папка с исходным кодом React)
2. ``npm i`` (установить зависимости)
3. ``npm run start`` (запустить в режиме разработчика)
4. ``npm run build`` (сгенерировать статичные файлы, чтобы потом хостить)
